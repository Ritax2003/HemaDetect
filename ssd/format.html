Detect Pulsars Using Pulsar Candidate Data

Returned by PDMP(part of the PSRCHIVE software package)

Content of .phcx file to be used as an Input:

  • Signal-to-Noise Ratio
  • Topocentric Period
  • Barycentric Period
  • Pulse Width
  • Dispersion Measure
  • Acceleration Value
  • Right Ascension (J2000) in Degrees
  • Declination (J2000) in Degrees
  • FFT Signal-to-Noise Ratio
  • Folded Profile
  • Candidate Sub-Integrations
  • Candidate Sub-Bands,
  • Dm_Curve[Tuple of arrays (DmValues, SnrValues)]
  • Accn_Curve[Tuple of arrays (AccnValues, SnrValues)]
  • Rank of the candidate within the beam it was found
  • Hits[Number of (DM, Acceleration) trial pairs]
  • Pdm_Plane[Tuple of arrays (PeriodCorrections, DmValues, SnrValues)]

    • Cited: https://astronomy.swin.edu.au/~vmorello/
