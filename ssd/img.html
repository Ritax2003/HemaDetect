Know Your Channels

Detect a Pulsar Using Images with the Following channels :

Channel 0: Period Correction - Dispersion Measure Surface:

  • Represents the dispersion measure surface, correcting for the interstellar medium's impact on pulse period.

    • Channel 1: Phase - Sub-band Surface:

  • Captures phase information across different frequency sub-bands, showing signal variation over the spectrum.

    • Channel 2: Phase - Sub-integration Surface:

  • Contains phase data over sub-integrations, showing how the signal varies across observation segments.

    • Input data format is similar as HTRU1 Dataset
    Cited: https://as595.github.io/HTRU1/
